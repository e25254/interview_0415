# Interview_0415

## How to start

1. change to `v16.13.1` node version

   ```shell
   nvm use v16.13.1
   # or
   nvm use
   ```

2. install `Vite` in global

   ```shell
   npm install -g vite
   ```

3. install plugin

   ```shell
   npm install
   ```

4. run the project

   ```shell
   npm run dev
   ```
